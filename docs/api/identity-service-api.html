Authentication Endpoints

Endpoints for user authentication, token management, and session control.

POST /auth/login
Authenticates a user with email and password, returns access and refresh JWT tokens. The access token should be used for subsequent API calls in the Authorization header.

Request Body

email string required
User's email address
password string required
User's password
tenantId string required
Tenant identifier for multi-tenant isolation

Example Request

{
  "email": "cashier1@coffee.test",
  "password": "Test@123456",
  "tenantId": "tenant-coffee-001"
}

Response Codes

200 OK

Login successful, returns access and refresh tokens

401 Unauthorized

Invalid credentials (wrong email or password)

423 Locked

Account locked due to multiple failed login attempts

Example Response (200 OK)

{
  "success": true,
  "data": {
    "accessToken": "eyJhbGciOiJIUzI1NiJ9...",
    "refreshToken": "eyJhbGciOiJIUzI1NiJ9...",
    "tokenType": "Bearer",
    "expiresIn": 86400,
    "user": {
      "id": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
      "tenantId": "tenant-coffee-001",
      "email": "cashier1@coffee.test",
      "firstName": "John",
      "lastName": "Doe",
      "phone": "+1234567890",
      "isActive": true,
      "emailVerified": true,
      "roles": ["CASHIER"],
      "permissions": ["products:read", "transactions:create"]
    }
  },
  "message": "Login successful"
}
POST /auth/refresh
Generates a new access token using a valid refresh token. Use this endpoint when the access token expires to obtain a new one without requiring the user to log in again.

Request Body

refreshToken string required
The refresh token obtained from login

Example Request

{
  "refreshToken": "eyJhbGciOiJIUzI1NiJ9..."
}

Response Codes

200 OK

Token refreshed successfully, returns new tokens

401 Unauthorized

Invalid or expired refresh token

GET /auth/validate üîí Requires Authentication
Validates a JWT token and returns user information if the token is valid. This endpoint can be used by frontend applications to verify authentication state and retrieve current user details.

Headers

Authorization string required
Bearer token: Bearer eyJhbGciOiJIUzI1NiJ9...

Response Codes

200 OK

Token is valid, returns user information

401 Unauthorized

Invalid or expired token

400 Bad Request

Authorization header is missing

POST /auth/logout üîí Requires Authentication
Logs out a user by invalidating the JWT token. After logout, the token can no longer be used for authentication. The frontend should remove the token from storage.

Headers

Authorization string required
Bearer token

Response Codes

200 OK

Logout successful

User Endpoints

Endpoints for user profile and account management.

GET /users/me üîí Requires Authentication
Retrieves the currently authenticated user's profile information including roles, permissions, and account details.

Headers

Authorization string required
Bearer token

Response Codes

200 OK

Returns user profile information

401 Unauthorized

Invalid or missing token

404 Not Found

User not found
