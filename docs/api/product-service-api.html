Product Endpoints

All product endpoints require Bearer authentication.

POST /products ðŸ”’ Requires Authentication
Creates a new product in the catalog. The SKU and barcode must be unique per tenant.

Request Body

{
  "code": "ESPRESSO",
  "name": "Espresso",
  "description": "Double shot espresso",
  "sku": "ESP-001",
  "barcode": "1234567890123",
  "categoryId": "uuid-of-category",
  "price": 25000.00,
  "cost": 15000.00,
  "taxRate": 10.0,
  "unit": "CUP",
  "isActive": true
}

Response Codes

201 Created

Product created successfully

400 Bad Request

Invalid input data

409 Conflict

Duplicate SKU or code
